How to Run Code:

*We've written custom scripts to make it easier to compile and run our code. Sequential is used in the examples, but replace script names appropriately.

Give permission to custom scripts:
  • chmod 755 c_seq r_seq

Compile source code using:
  • ./c_seq

Run using:
  • ./r_seq [size] [nthreads (openMP only)]

*size = desired matrix size, nthreads = number of threads to use To change MPI ntasks, edit mpi_slurm.sh first
